# TypescriptBundle

idea come from https://github.com/laravel/wayfinder

## Features

- Generate TypeScript interfaces from PHP classes and interfaces
- Support for PHP 8.2 and above
- Customizable configuration options
- Integration with Symfony

## Installation

- Composer: `composer require your-vendor/typescript-bundle`
- Symfony Flex: `composer require your-vendor/typescript-bundle --dev`
